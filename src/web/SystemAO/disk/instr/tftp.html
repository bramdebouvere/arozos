TFTP Protocol Notice

TFTP (Trivial File Transfer Protocol) is a simple, lightweight file transfer protocol designed for basic file transfer operations.

  • No Authentication: TFTP does not support user authentication. Access control is managed through the default user configuration.
  • No Directory Listing: TFTP clients cannot browse directories. You must know the exact file path.
  • File Size Limit: Maximum file size is 32MB per transfer.
  • UDP Protocol: TFTP uses UDP (port 69 by default), which may require firewall configuration.
  • Security Warning: Use TFTP only in trusted networks. For secure file transfer, consider using SFTP instead.
Using TFTP on Windows

Windows includes a built-in TFTP client. First, enable it via Control Panel:

  1. Open Control Panel â†’ Programs and Features
  2. Click Turn Windows features on or off
  3. Check TFTP Client and click OK

To download a file from ArozOS TFTP server, open Command Prompt and run:

tftp -i [hostname] GET [filesystem-uuid]/path/to/file.txt C:\destination\file.txt

To upload a file to ArozOS TFTP server:

tftp -i [hostname] PUT C:\source\file.txt [filesystem-uuid]/path/to/file.txt
Using TFTP on macOS

macOS includes a built-in TFTP client accessible via Terminal.

To download a file from ArozOS TFTP server, open Terminal and run:

tftp [hostname]
get [filesystem-uuid]/path/to/file.txt /destination/path/file.txt
quit

To upload a file to ArozOS TFTP server:

tftp [hostname]
put /source/path/file.txt [filesystem-uuid]/path/to/file.txt
quit
Using TFTP on Linux

Install TFTP client if not already installed:

sudo apt-get install tftp-hpa (Debian/Ubuntu)
sudo yum install tftp (RedHat/CentOS)

To transfer files:

tftp [hostname]
get [filesystem-uuid]/path/to/file.txt
put /local/file.txt [filesystem-uuid]/path/to/file.txt
quit
File System UUID Path Format

TFTP file paths in ArozOS follow this format:

[filesystem-uuid]/path/to/file

To find your filesystem UUID, check your File Manager or system settings. Common UUIDs include:

  • user - Your personal user directory
  • tmp - Temporary files directory

You can use either: user:/documents/report.txt or /user/documents/report.txt in TFTP
